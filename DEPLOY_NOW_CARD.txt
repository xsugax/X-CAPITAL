Q: Do I need to be a tech expert?
A: No, just copy-paste the deployment command. The script handles everything.

Q: What if DNS isn't propagated?
A: Can take 24 hours. Deployment is ready, just wait for DNS before testing.

Q: What if I don't have SSL yet?
A: Use Let's Encrypt (free, 1 minute). Script checks for it and asks if you want to continue.

Q: Can I use a different domain?
A: Yes, change xcapital.investments in:
   • backend/.env (FRONTEND_URL)
   • frontend/.env.local (NEXT_PUBLIC_API_URL, NEXT_PUBLIC_WS_URL)
   • nginx.conf (server_name)

Q: How do I monitor after deployment?
A: docker compose -f docker-compose.prod.yml logs -f

Q: How do I stop/restart?
A: docker compose -f docker-compose.prod.yml down
   docker compose -f docker-compose.prod.yml up -d

Q: Is my data safe?
A: Yes. PostgreSQL and Redis use persistent Docker volumes.
   Backup regularly with: docker compose exec postgres pg_dump -U xcapital xcapital_db
